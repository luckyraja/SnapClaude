# Meta Decks Command

Search the web for the best Marvel Snap meta decks currently dominating the competitive scene.

## What to search for:
- Current Marvel Snap tier list and meta decks (December 2025)
- Top performing decks from recent tournaments
- Popular deck archetypes and their card lists
- Win rates and popularity statistics

## How to use User's data
- Look at the CSV that starts with marvel_snap_collection
- Analyze this CSV and understand that it is the user's current card collection 

## What to provide:
1. Top 5 decks with full card lists
2. Tier rankings (S-tier, A-tier, etc.)
3. list the top 5 decks that can be built with the user's current card collection
4. list the top 5 decks that can be built where the user is missing one required card. 
5. Recent meta shifts or new strategies
6. How these decks compare to my current collection (if available)
