# Marvel Snap Collection Optimizer
The goal of this project is to understand the onlne card game Marvel Snap. Marvel snap is a mobile card game. The game updates frequently, it is important to speak with context of recent marvel snap releases or annoucments. 


# User Goals
The goals of this project is to help the user, raja, built a competitive and fun to play marvel snap deck. An added benefit of this project is that raja will learn different aspects of basic software and web development. 

I'm working on building the best Marvel Snap collection by:
- Analyzing my current card collection, it is present in our workdirectory as a csv 
- Figuring out what cards to roll for
- Forecasting what cards will be better in the future
