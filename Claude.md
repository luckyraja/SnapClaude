# SnapClaude Project

## Project Goal
Building the best Marvel Snap collection by making data-driven decisions about card acquisition and meta predictions.

## Core Objectives

### 1. Collection Optimization
- Track and maintain comprehensive card collection data
- Identify gaps in collection and prioritize card acquisition
- Analyze collection completeness across different series and archetypes

### 2. Card Roll Strategy
- Determine which cards to roll for based on:
  - Current meta strength
  - Versatility across deck archetypes
  - Synergy with existing collection
  - Series drop value and acquisition cost

### 3. Future Meta Forecasting
- Predict which cards will increase in value
- Identify upcoming meta shifts before they happen
- Analyze patch notes and balance changes for strategic advantage
- Track card usage rates and win rates from community data
